# Car-Rental-Management-System
Car Rental Management System(GUI+DATABASE) using Java for backend, JavaSwing for User interface and SQL as Database
The Car Rental Management System is a desktop-based application developed using Java Swing for the graphical user interface and SQL for backend database management.
The system automates car rental operations such as customer registration, car booking, return processing, and billing, reducing manual effort and improving accuracy.
This project demonstrates the use of Java GUI programming, database connectivity (JDBC), and CRUD operations, making it suitable for academic submission and internships.

Technologies Used
1 Java
2 Java Swing
3 MySQL Database
4 JDBC
5 MVC Architecture

Key Features
1 Secure login system using email and password
2 Admin and Client role-based access
3 Create new user accounts
4 SQL-based persistent storage
5 User-friendly graphical interface
6 Input validation and error handling

Project Structure
The project is organized using the MVC pattern with separate Controller and Model packages. The Controller
handles user interaction and navigation, while the Model manages database connectivity and business logic.

Database
The system uses a SQL database with a users table that stores user details such as name, email, password,
phone number, and role type (Admin or Client).

Execution
To run the application, configure the database connection in the Database class and execute the Main class
located in the Controller package.
